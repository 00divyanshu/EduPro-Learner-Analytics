# EduPro Learner Analytics Final Report

## 1. Introduction
EduPro is an online learning platform serving learners across multiple demographics, learning goals, and subject interests. This project analyzes learner demographics and course enrollment behavior using descriptive analytics.

## 2. Methodology
The project uses three datasets: users, courses, and transactions. The workflow includes data loading, schema validation, cleaning, merging, feature engineering, KPI creation, EDA, visualization, and business insight generation.

## 3. Data Preparation
The pipeline standardizes column names, cleans demographic fields, parses transaction dates, removes duplicate records, validates required columns, and creates an integrated master analytics table.

## 4. Analytics Framework
The analysis focuses on:
- demographic participation
- course category popularity
- course level preference
- monthly enrollment trends
- beginner vs advanced learner behavior

## 5. Dashboard
The Streamlit dashboard provides interactive filters, KPI cards, Plotly charts, business insights, recommendations, and a preview of filtered records.

## 6. Recommendations
Final recommendations should be updated after real data is loaded. The project already includes a recommendation generator that highlights category demand, learner progression, and inclusivity opportunities.

## 7. Conclusion
This project demonstrates an industry-style descriptive analytics workflow suitable for data analyst, AI/ML internship, and analytics portfolio presentation.
