# Interview Explanation Notes

## One-Minute Explanation
This project is a descriptive learner analytics system for EduPro, an online learning platform. I used users, courses, and transaction datasets to analyze learner demographics, course popularity, course-level preference, and enrollment behavior. I built a modular Python pipeline and an interactive Streamlit dashboard to convert raw data into business insights.

## Why This Project Matters
It helps an education platform understand which learner groups are most active, what courses they prefer, and how course strategy can be improved using data.

## Technical Talking Points
- Modular project structure with separate cleaning, merging, feature engineering, KPI, visualization, and insight modules.
- Schema validation and custom exceptions for reliable data processing.
- Logging for debugging and production-style traceability.
- Streamlit dashboard with interactive filters and Plotly charts.
- Descriptive analytics focus instead of unnecessary prediction modeling.

## Resume Bullet
Built an end-to-end learner analytics dashboard using Python, Pandas, Plotly, and Streamlit to analyze demographic patterns, course popularity, enrollment trends, and learner behavior across multiple integrated datasets.
