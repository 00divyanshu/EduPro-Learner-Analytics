# Executive Summary

## Project
Learner Demographics and Course Enrollment Behavior Analysis on EduPro

## Business Goal
EduPro needs to understand who its learners are, which courses they prefer, and how enrollment behavior varies across demographics. This project turns learner, course, and transaction records into actionable insights for course planning, engagement, inclusivity, and targeted learning strategy.

## Key Questions
- Which age groups are most active?
- How do enrollments differ by gender?
- Which course categories and levels are most popular?
- How do beginner and advanced learner behaviors differ?
- What actions can improve learner engagement and course strategy?

## Expected Business Value
- Better course planning using category and level demand.
- More targeted learner engagement by age group and gender.
- Stronger inclusivity monitoring across demographic segments.
- Clear portfolio-ready dashboard for decision-making.
