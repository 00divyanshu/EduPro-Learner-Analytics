# Learner Demographics and Course Enrollment Behavior Analysis on EduPro

## Abstract
This study analyzes learner demographics and course enrollment behavior on EduPro, an online learning platform. The objective is to identify meaningful learner segments, course preferences, and enrollment trends that can support educational planning and learner engagement strategy.

## Keywords
Learner analytics, descriptive analytics, online education, enrollment behavior, demographic analysis, Streamlit dashboard

## 1. Introduction
Online learning platforms generate valuable behavioral data through learner registrations, course catalogs, and enrollment transactions. Understanding this data helps platforms design better learning experiences and improve decision-making.

## 2. Objectives
The study aims to analyze learner activity by age group, gender, course category, course type, and course level. It also examines beginner and advanced learner behavior to support better course planning.

## 3. Data and Methods
The project uses users, courses, and transactions datasets. Data preprocessing includes cleaning, validation, integration, and feature engineering. Descriptive statistics and interactive visualizations are used to interpret learner behavior.

## 4. Expected Findings
The analysis is expected to identify the most active learner groups, high-demand categories, preferred course levels, and periods of high enrollment activity.

## 5. Practical Implications
EduPro can use these insights to improve course offerings, personalize campaigns, monitor inclusivity, and guide learners across progressive course levels.

## 6. Conclusion
The project provides a practical learner intelligence system using professional data analytics architecture and dashboard deployment practices.
