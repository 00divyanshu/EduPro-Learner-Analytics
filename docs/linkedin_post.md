# LinkedIn Project Post Draft

I built an end-to-end learner analytics project titled **Learner Demographics and Course Enrollment Behavior Analysis on EduPro**.

The project analyzes user demographics, course catalog data, and enrollment transactions to understand learner behavior across age groups, gender, course categories, and course levels.

Key highlights:
- Built a reusable Python data pipeline
- Cleaned and integrated multiple datasets
- Created learner and course-level analytics features
- Designed KPIs for enrollment intelligence
- Developed an interactive Streamlit dashboard
- Prepared business insights and recommendations
- Structured the repository for GitHub portfolio presentation

Tools used: Python, Pandas, NumPy, Plotly, Streamlit, OpenPyXL, PyYAML, Pytest

This project strengthened my understanding of descriptive analytics, dashboard design, business storytelling, and professional project structuring.
